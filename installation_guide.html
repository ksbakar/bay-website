R and RStudio

Have R and RStudio installed and working

It is important that everyone has R and RStudio installed and functional before any assignments are due.

How to install R?

The latest version of R can be downloaded from the Comprehensive R Archive Network (CRAN):

https://cran.r-project.org/

To install R on a PC click on “Download R for Windows”

https://cran.r-project.org/bin/windows/base/

and then “install R for the first time”. This will take you to a new window with the latest version of R. Click on “Download R 4.X.X for Windows” and follow the installation instructions. Default settings are OK. Note: At the time of writing, the current version of R was 4.2.2 but you can install whichever is the latest version, even if it differs from the one mentioned in the course notes.

To install R on a Mac click on “Download R for (Mac) OS X”

https://cran.r-project.org/bin/macosx/

and download the R binary under “Latest release”. The file will have an extension of .pkg, for example R-4.X.X.pkg. Click on this link and follow the instructions. Default settings are OK. Note: you can install whichever is the latest version, even if it differs from the one mentioned in these notes.

Optional – Check that R has installed correctly (You can skip this step and proceed to installing RStudio. If RStudio is working then you know that base R was successfully installed.) Open R and type 1 + 2 into the command line (the command line is indicated with the “>” symbol). After pressing Enter, you should get the following output:

[1] 3

How to install RStudio?

You don’t need to download RStudio in order to use R, but RStudio provides a more user friendly interface and is commonly used by many R users. We will be using RStudio, so we strongly recommend that you install it too.

First make sure you have downloaded base R and confirmed that it was successfully installed. After you have downloaded base R, you can download RStudio. RStudio can be downloaded from:

https://posit.co/download/rstudio-desktop/#download

Scroll down and download the appropriate file for your operating system. Follow the prompts to install.

To confirm that RStudio has installed correctly, double click on the RStudio icon your computer to open the program

If you type 1 + 2 into the R Console and hit enter you should get 3 as the output.

Good work! You have now successfully installed R and RStudio!

How to Install Compiler?

Install ‘Rtools’ (C++ Compiler) for Windows OS

Download Rtools from

https://cran.r-project.org/bin/windows/Rtools/

based on the R version that you have already installed. Usually, for the latest version this should be RTools 4.4, i.e.,

https://cran.r-project.org/bin/windows/Rtools/rtools44/rtools.html

For further detail on Rtools see:

https://github.com/stan-dev/rstan/wiki/Configuring-C—Toolchain-for-Windows

Option 1: Install ‘Rtools’ for Mac OS

Install ‘macrtools’ R package. This installer package was developed by James Joseph Balamuta, see details:

https://mac.thecoatlessprofessor.com/macrtools/

According to their webpage: “This package is designed to recreate the compiled code toolchain used to compile the official macOS R binary on CRAN by following the steps described on the r-project developer page for macOS tools (https://mac.r-project.org/tools/). The package is able to to setup the compilation toolchain on any Mac that meets the standards required to install the official CRAN R binary on macOS.”

You can install the development version of macrtools from GitHub using R console or using “Tools” menu.

First install ‘remotes’ package (if not already installed)

install.packages(“remotes”)

Then install ‘macrtools’ packages:

remotes::install_github(“coatless-mac/macrtools”)

Then using ‘macrtools’ package install rtools as:

macrtools::macos_rtools_install()

This will install: (1) Xcode CLI (2) gfortran, and (3) R Development binaries from the Recipes project.

For further detail on installation see:

Option 2: Install Compiler for Mac OS

First, if you have the files ~/.R/Makevars and/or ~/.Renviron, save a copy in a different location (like your desktop) and delete them.

If you don’t know what these files are or if you have them, go to Finder > Go > Home, which takes you to your home directory, and press SHIFT + CMD + . to show hidden files (i.e. files starting with a full stop .) if you can’t see them already. Now search among the hidden files and if you see a folder named .R/ and/or a file .Renviron, copy them to your desktop and delete the original copies from your home directory.

To install the Xcode Command Line Tools, run the following in the Terminal (open Finder > Applications > Terminal, type in the following command and press ENTER):

xcode-select –install

macOS will download and install the Xcode CLT (it will take a while, make sure you have a stable internet connection).

If your Mac has an Intel chip, you need to install gfortran v8.2 (for Mojave) independent of your macOS version. You can download the installer from

https://github.com/fxcoudert/gfortran-for-macOS/releases/tag/8.2

If your Mac has an Apple M1 chip, you will need to install gfortran v11. You can download it from here:

https://github.com/fxcoudert/gfortran-for-macOS/releases/tag/11-arm-alpha2

download the .pkg package listed under Assets at the bottom of the page.

Install ‘rstan’ and ‘brms’ packages

‘rstan’ package

– After installing Rtools (Windows and/or Mac OS), open RStudio and using R console or using “Tools” menu install ‘rstan’ package.

install.packages(“rstan”, dependencies = TRUE)

For detail on ‘rstan’ installation:

https://github.com/stan-dev/rstan/wiki/RStan-Getting-Started

Now, verify the installation by running:

example(stan_model, package = “rstan”, run.dontrun = TRUE)

in your R console.

‘brms’ package

– Then install ‘brms’ package:

install.packages(“brms”)

Great! Now you are ready to use Stan!
